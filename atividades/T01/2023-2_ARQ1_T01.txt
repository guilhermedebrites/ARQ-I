ASIC (Application-Specific Integrated Circuit): Um circuito integrado projetado para uma função específica.

ASSP (Application-Specific Standard Product): Um circuito integrado pré-projetado para uma aplicação específica em um mercado.

SPLD (Simple Programmable Logic Device): Um dispositivo de lógica programável para funções lógicas simples.

CPLD (Complex Programmable Logic Device): Um dispositivo de lógica programável para circuitos digitais complexos com capacidade de reprogramação.

SoC (System on a Chip): Um circuito integrado que incorpora todos os componentes de um sistema eletrônico em um único chip.

FPGA (Field-Programmable Gate Array): Um circuito integrado reprogramável com blocos de lógica programável e interconexões configuráveis.
